What is NovaHoyaAI?

NovaHoyaAI is an advanced edge-computing intelligence framework engineered for high-speed, real-time deep-learning execution on mobile and IoT devices. It uses modular JSON pipelines, optimized hardware acceleration, and pure device-local inference.
